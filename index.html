翻译设置

使用 DeepL 免费 API 可获得更高质量的翻译，免费额度 50 万字符/月。需到 DeepL 官网注册获取 API 密钥。

密钥仅保存在本浏览器（localStorage），不会上传至任何服务器。留空则使用默认免费翻译 MyMemory。

部署到 CloudStudio 等公网环境时，服务器需设置环境变量 DEEPL_PROXY_TOKEN，此处填入相同的值，否则代理会拒绝翻译请求（防止他人盗用你的 DeepL 额度）。
